Typora 用法

Typora的语法

1 格式

表格
可以在—分割线加入：设置表格对齐方式
“:——–“表示左对齐，”:———:“表示居中对齐，”———-:”表示右对齐

无序列表：- + 空格
有序列表：1. + 空格
引用：> + 空格

分割线: ---回车即可

引用 md 文件内部内容：

  • [需要跳转的文本](#destination)

  • <a id="destination">正常显示的文本</a>

使用上述语法，即可通过需要跳转的文本跳转到正常显示的文本 例如 表情

高亮:

  • ==高亮文本== typora需要偏好设置里面开启一下
  • <mark>highlight 2</mark>

复选框：

  • 未完成的任务
  • 已完成的任务
  • 另一个未完成的任务

此处typora 需要先写- 后面的内容 因为 - 空格直接就变成无序了

注解：

创建脚注格式类似这样 [^RUNOOB]。

[^RUNOOB]: 菜鸟教程 – 学的不仅是技术，更是梦想！！！

2、快捷键

标题：ctrl + 数字
表格：ctrl + t

表格增加一行快捷键：选择行数 按下ctrl+enther

生成目录：[TOC] + 回车
选中一整行：ctrl + l
选中单词：ctrl + d
选中相同格式的文字：ctrl + e
跳转到文章开头：ctrl + home
跳转到文章结尾：ctrl + end
搜索：ctrl + f
替换：ctrl + h
加粗：ctrl + b
倾斜：ctrl + i
下划线：ctrl + u
删除线：alt + shift + 5
插入图片：直接拖动到指定位置即可或者ctrl + shift + i
插入链接：ctrl + k
