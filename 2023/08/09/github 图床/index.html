github 图床

1、配置Github

进入github平台 github

登录账号 创建仓库

配置仓库名 勾选 Add a README file 点击create

点击头像 创建token 界面划到最下面 点击<>Developer settings 设置token永不过期

创建完保存好 仅可见一次

2、配置PicGo

官网网站下载 PicGo

安装完成之后配置Github图床配置

设定仓库名：Github用户名/仓库名

设定分支名：main

设定Token: 前面保存的token信息

设定存储路径：img/

设定自定义域名：使用cdn加速域名

https://cdn.staticaly.com/gh/gondmhd/blogImage@main

3、配置typora

文件-偏好设置-图像设置图床配置

外观 自定义字段大小

3.1 typora颜色样式

==蓝色==
==橘色==
==红色==

4、配置Pandoc

Pandoc下载地址

将解压安装路径配置window环境变量path

pandoc –version # 检查是否生效

重启Typora
